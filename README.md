# 🌌 GalaxyShop — Wear the Universe

Site de prezentare pentru un shop de haine cu tematică galaxie/spațiu, dezvoltat în echipă cu React, Tailwind CSS și Strapi CMS.

**Echipa:** Lucian, Lorena, Nikolaus

## 🔗 Linkuri

- **Site Netlify:** https://galaxyshop.netlify.app
- **Strapi App:** https://galaxyshop.strapiapp.com
- **Tema tweakcn:** https://tweakcn.com/editor/theme

## 👥 Echipa

| Membru   | Sarcini                                                                     |
| -------- | --------------------------------------------------------------------------- |
| Lucian   | Setup proiect, Navbar, Footer, dark mode, pagina Home, Shop, About, Contact |
| Lorena   | Testare, review cod, deployment Netlify                                     |
| Nikolaus | Configurare Strapi CMS, conținut pagini                                     |

## 📄 Pagini

- **Home** — Hero Banner cu galaxie, categorii, produs featured
- **Shop** — Grid produse cu filtre pe categorii
- **About** — Single type, echipa, valorile brandului
- **Contact** — Formular de contact fără bază de date

## 🛠️ Tehnologii

- React 18 + TypeScript
- Tailwind CSS v4
- React Router DOM
- Vite
- Strapi CMS (Headless)
- Netlify (Hosting)

## 🖼️ Fonturi

- **Orbitron** — titluri și logo
- **Rajdhani** — meniu și butoane
- **Inter** — text general
